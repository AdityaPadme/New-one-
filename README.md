# Paradise Nursery Shopping Application

## About the Project
Paradise Nursery is an online plant shop built with React. This application allows users to browse a variety of beautiful houseplants, view details like pricing and images, add them to a shopping cart, and manage their cart items. It features a dynamically updating cart and calculates total costs as items are added, removed, or adjusted.

## Features
* **Landing Page:** A welcoming home page to get started.
* **Product Listing:** View unique houseplants categorized for easy browsing.
* **Shopping Cart:** Add plants, increase/decrease quantities, or remove items.
* **Dynamic Updates:** The cart icon and total prices update automatically.

## Technologies Used
* React
* Redux Toolkit (for state management)
* React Router (for navigation)
* CSS

## How to Run This Project Locally
1. Clone the repository.
2. Open your terminal and run `npm install` to install dependencies.
3. Run `npm run dev` to start the local server.
4. Open the provided `localhost` link in your browser to view the app.
